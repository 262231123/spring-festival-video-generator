# 🧧 春节拜年视频生成器

一个基于 Next.js 的AI春节拜年视频生成应用，让用户上传照片，选择风格，生成专属春节拜年视频。
公网测试地址：47.79.249.163

## ✨ 功能特点

- 📸 **照片上传：支持上传用户个人照片
- 🎨 **多种风格**：提供 5 种春节拜年风格
  - 财神到
  - 舞狮拜年
  - 阖家团圆
  - 烟花秀
  - 挥春送福
- 🎬 **视频生成**：基于 Seedance 1.5 生成 5 秒视频
- 💬 **幽默台词**：每个风格都配有幽默的春节台词
- 📥 **视频下载**：支持下载生成的视频

## 🚀 快速开始

### 安装依赖：

```bash
cd spring-festival-video-generator
npm install
```

### 启动开发服务器：

```bash
npm run dev
```

打开 [http://localhost:3000](http://localhost:3000) 查看应用。

### 构建生产版本：

```bash
npm run build
npm start
```

## 🎯 使用说明

1. **上传照片**：点击上传区域或拖拽照片到上传区域
2. **选择风格**：从 5 种春节风格中选择一个
3. **查看预览**：可以看到选中风格的动作和台词
4. **生成视频**：点击"生成拜年视频"按钮
5. **等待完成**：视频生成大约需要 1-3 分钟
6. **下载视频**：生成完成后可以预览和下载视频

## 🎨 拜年风格

### 财神到
- 描述：扮演财神爷，手捧金元宝，喜气洋洋
- 动作：双手捧着金元宝向观众展示
- 台词："恭喜发财！红包拿来！"

### 舞狮拜年
- 描述：舞狮表演，热闹喜庆
- 动作：模仿狮子摇头晃脑，做出拜年的动作
- 台词："新春快乐！万事如意！"

### 阖家团圆
- 描述：温馨家庭聚餐，共享年夜饭
- 动作：举杯向家人和观众敬酒祝福
- 台词："新年快乐！身体健康！"

### 烟花秀
- 描述：绚丽烟花下的浪漫祝福
- 动作：仰望烟花，双手合十许下新年愿望
- 台词："愿新年，胜旧年！"

### 挥春送福
- 描述：写春联送祝福，书香门第
- 动作：写下福字，然后向观众展示
- 台词："福到啦！吉祥如意！"

## 🔧 技术栈

- **Next.js 14 - React 框架
- **TypeScript** - 类型安全
- **Tailwind CSS** - 样式框架
- **Axios** - HTTP 客户端
- **Seedance 1.5 API** - 视频生成

## 📝 注意事项

- 确保网络连接正常，视频生成需要调用外部 API
- 视频生成时间取决于网络和 API 响应速度
- 建议使用清晰的人脸照片以获得更好的效果
